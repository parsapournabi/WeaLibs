Detailed Description

Abstract series class (it can't be create as object). Creation idea of this class is: Because the GLSeriesStorage is a template class and it can't inherit from Qt Objects. GLSeriesStorage can't have any signal or emit any signal. So this GLAbstractSeries class created to GLSeriesStorage class inherit from it. Now GLSeriesStorage class can have signals and also can have some method to emit signals. (like requestUpdateSeriesData())

Note
All methods are virtual required (that means it must override at inherited class.)
