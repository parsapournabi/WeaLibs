Detailed Description

Contains the minX, maxX, minY, maxY properties, which define the limits of the Chart XY axes.

PropertyAxis Range will use on GLChartView class property. This is much cleaner than using 4 properties(minX, maxX, minY and ...) instead of a single property at GLChartView class.
