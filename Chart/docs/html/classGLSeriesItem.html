Detailed Description

GLSeriesStorage class as QML Component.

GLSeriesItem

This class is used n QML SeriesItem. It is similar to the GLSeriesStorage class but has some differences.

Note
The series property is a required property that must be set in QML. By setting the series property, all signals and properties of the series object will synchronized with this class's signals/properties.
